How to Set Up Your Email Header Image

Option 1: WordPress Customizer (Recommended)

  1. Go to Appearance â†’ Customize in your WordPress admin
  2. Click on Header Image
  3. Upload or select your logo/header image
  4. Save changes

Option 2: Direct File Upload

  1. Upload your header image to /wp-content/uploads/email-header.jpg
  2. The image should be approximately 400x200px for optimal display
  3. Use JPG or PNG format

Option 3: Customize in Code

You can modify the email template to use a different image path:

// In functions/email-templates.php, line ~30 $header_image_url = get_header_image() ?: $site_url . '/wp-content/uploads/email-header.jpg'; // Change to your preferred image: $header_image_url = $site_url . '/wp-content/uploads/your-custom-image.jpg';

Image Requirements

  • Recommended size: 400x200 pixels
  • Format: JPG, PNG, or WebP
  • File size: Under 200KB for fast email loading
  • Background: Transparent or light background works best
